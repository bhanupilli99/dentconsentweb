DentConsent API

Base URL: http://10.161.178.54:8000/api/

Authentication

/login POST

User login with email and password.

curl -X POST http://10.161.178.54:8000/api/login \ -H "Content-Type: application/json" \ -d '{"email": "user@example.com", "password": "password123"}'
/register POST

Register a new patient or doctor account.

curl -X POST http://10.161.178.54:8000/api/register \ -F "email=tester@dentconsent.com" \ -F "password=secret" \ -F "full_name=Test User" \ -F "role=patient" \ -F "mobile_number=1234567890" \ -F "image=@/path/to/profile.jpg"
/send_otp POST

Send OTP for registration or password reset.

curl -X POST http://10.161.178.54:8000/api/send_otp \ -H "Content-Type: application/json" \ -d '{"email": "user@example.com", "action": "Registration"}'
/verify_otp POST

Verify the OTP received via email.

curl -X POST http://10.161.178.54:8000/api/verify_otp \ -H "Content-Type: application/json" \ -d '{"email": "user@example.com", "otp": "1234", "action": "Registration"}'

User Profiles

/update_profile_image POST

Upload or update a user's profile picture.

curl -X POST http://10.161.178.54:8000/api/update_profile_image \ -F "user_id=1" \ -F "image=@/path/to/profile.jpg"
/update_doctor_signature POST

Upload a doctor's digital signature for consent forms.

curl -X POST http://10.161.178.54:8000/api/update_doctor_signature \ -F "user_id=1" \ -F "image=@/path/to/signature.png"

Education content

/get_education_content GET

Fetch benefits, risks, and topics for a procedure.

curl -X GET "http://10.179.235.54:8000/api/get_education_content?operation_type_id=1"
/get_procedure_steps GET

Fetch step-by-step procedure guide and videos.

curl -X GET "http://10.179.235.54:8000/api/get_procedure_steps?operation_type_id=1"
/get_quiz GET

Retrieve quiz questions for procedure validation.

curl -X GET "http://10.179.235.54:8000/api/get_quiz?operation_type_id=1&language=en"
/submit_consent_quiz POST

Submit patient quiz results for a treatment record.

curl -X POST http://10.179.235.54:8000/api/submit_consent_quiz \ -H "Content-Type: application/json" \ -d '{ "treatment_id": 1, "quiz_score": 5, "total_questions": 5, "quiz_data": [ {"question_text": "Is this safe?", "selected_option": "Yes", "is_correct": true} ] }'

Treatments

/get_treatments GET

Get clinical treatments for a patient or doctor.

curl -X GET "http://10.179.235.54:8000/api/get_treatments?user_id=1&role=patient"
/create_treatment POST

Doctor creates a new treatment plan for a patient.

curl -X POST http://10.179.235.54:8000/api/create_treatment \ -H "Content-Type: application/json" \ -d '{"doctor_id": 1, "patient_id": 2, "operation_type_id": 3, "clinical_notes": "Urgent"}'
/get_operation_types GET

List all available dental procedure types.

curl -X GET "http://10.179.235.54:8000/api/get_operation_types"

Messaging

/get_conversations GET

Retrieve chat history overview for a user.

curl -X GET "http://10.179.235.54:8000/api/get_conversations?user_id=1"
/send_message POST

Send a direct message to another user.

curl -X POST http://10.179.235.54:8000/api/send_message \ -H "Content-Type: application/json" \ -d '{"sender_id": 1, "receiver_id": 2, "message": "Hello"}'

Digital Consent

/get_consents GET

List signed consent forms for a patient.

curl -X GET "http://10.179.235.54:8000/api/get_consents?patient_id=2"
/submit_patient_signature POST

Finalize consent by uploading a digital signature.

curl -X POST http://10.179.235.54:8000/api/submit_patient_signature \ -H "Content-Type: application/json" \ -d '{"treatment_id": 1, "signature_base64": "data:image/png;base64,...", "is_confirmed": true}'
/serve_consent_pdf GET

Download the generated consent PDF.

curl -X GET "http://10.179.235.54:8000/api/serve_consent_pdf?treatment_id=1" --output consent.pdf

AI Assistant

/ai_chat POST

Chat with DentConsent AI for dental guidance.

curl -X POST http://10.179.235.54:8000/api/ai_chat \ -H "Content-Type: application/json" \ -d '{"message": "What is an implant?", "role": "patient"}'
